Class: IdCard

composer-common. IdCard

Business Network Card. Encapsulates credentials and other information required to connect to a specific business network
as a specific user.

Instances of this class can be created using IdCard.fromArchive or IdCard.fromDirectory, as well as the constructor.

new IdCard(metadata, connectionProfile)

Business Network Card to connect to a business network

Create the IdCard.

Parameters:
Name Type Description
metadata Object

metadata associated with the card.
connectionProfile Object

connection profile associated with the card.
