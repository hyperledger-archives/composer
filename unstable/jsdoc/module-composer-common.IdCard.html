Class: IdCard

composer-common. IdCard

An ID card. Encapsulates credentials and other information required to connect to a specific business network
as a specific user.


Instances of this class should be created using IdCard.fromArchive.

new IdCard(metadata, connectionProfile, credentials)

Create the IdCard.


Note: Only to be called by framework code. Applications should
retrieve instances from IdCard.fromArchive

Parameters:
Name Type Description
metadata Object

metadata associated with the card.
connectionProfile Object

connection profile associated with the card.
credentials Object

credentials used to connect to business network.
