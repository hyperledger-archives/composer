Class: ParticipantRegistry

composer-client. ParticipantRegistry

The ParticipantRegistry is used to manage a set of participants stored on the blockchain.

new ParticipantRegistry(id, name, securityContext, modelManager, factory, serializer)

Create an participant registry.


Note: Only to be called by framework code. Applications should
retrieve instances from BusinessNetworkConnection

Parameters:
Name Type Description
id string

The unique identifier of the participant registry.
name string

The display name for the participant registry.
securityContext SecurityContext

The security context to use for this participant registry.
modelManager ModelManager

The ModelManager to use for this participant registry.
factory Factory

The factory to use for this participant registry.
serializer Serializer

The Serializer to use for this participant registry.
