Class: HFCConnection

(protected) HFCConnection(connectionManager, connectionProfile, businessNetworkIdentifier, chain, connectOptions)

Class representing a connection to a business network running on Hyperledger
Fabric, using the hfc module.

The name of the connection profile associated with this connection
businessNetworkIdentifier string

The identifier of the business network for this connection,
or null if this connection if an admin connection
chain hfc.Chain

A configured and connected hfc.Chain object.
connectOptions object

The connection options in use by this connection.
