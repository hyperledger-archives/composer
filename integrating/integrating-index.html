Integrating existing systems

Fabric Composer can be integrated with existing systems by using a Loopback API. Integrating existing systems allows you to pull data from already existing business systems and convert it to assets or participants in a Composer business network.

For instructions on setting up a Loopback API, see Generating a REST API.
